#  Student Feedback Management System

## Project Overview
The Student Feedback Management System is a web-based application developed to collect, manage, and analyze student feedback efficiently.

It provides an interactive interface for students to submit feedback and for administrators to review and manage responses securely.


## Features
- Student feedback submission
- Admin dashboard
- Feedback management
- Secure data storage
- User-friendly interface


## Technologies Used
- PHP
- MySQL
- HTML
- CSS
- JavaScript
- XAMPP Server
  

## Setup Instructions
1. Install XAMPP
2. Move project folder to `htdocs`
3. Import `feedback.sql` into phpMyAdmin
4. Start Apache & MySQL
5. Open:http://localhost/Student-Feedback-System


##  Objective
To simplify feedback collection and analysis in educational institutions using a web-based solution.
